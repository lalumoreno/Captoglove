◆ SetConnectionIntervalAsync()

Task GSdkNet.Board.IBoardPeripheral.SetConnectionIntervalAsync ( float  value)

Sets connection interval. Values less that 7.5 ms will be clamped to 7.5 ms. This interval defines glove - OS average connection time. Smaller interval tends to faster connection but drains glove battery.

Parameters
valueConnection interval, milliseconds
Returns
Asynchronous task

Throws on connection or other failure.
